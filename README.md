seed-color
-------------------------------

[![][npm-badge]][npm-link]
[![][mit-badge]][mit]

## Install
``` shell
$ npm install seed-color --save
```

## Example
``` javascript
const seedColor = require('seed-color');

seedColor('hima').toHex(); // #d72c5d
seedColor('saku').toHex(); // #ae6e6c
```
